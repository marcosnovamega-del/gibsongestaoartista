# ⚡ Guia Rápido - Gibson Manager Pro

## 🚀 Início Rápido (5 minutos)

### 1️⃣ Primeiro Acesso

Abra `index.html` no navegador e faça login:

```
👤 Usuário: admin
🔐 Senha: admin123
```

### 2️⃣ Explore o Dashboard

Você verá:
- ✅ 3 eventos cadastrados
- ✅ KPIs do mês atual
- ✅ Gráfico de receita
- ✅ Próximos eventos

### 3️⃣ Veja os Artistas

Clique em **Artistas** no menu lateral:
- 4 artistas já cadastrados
- Clique em um artista para ver o perfil completo
- Explore as abas: Eventos, Equipe, Contratos, Modelo

### 4️⃣ Cadastre um Novo Evento

1. Vá em **Eventos** → **Novo Evento**
2. Preencha os dados:
   - Selecione o artista
   - Data e local do show
   - Dados do contratante
   - Cachê e comissão
3. Clique em **Salvar**
4. O contrato é gerado automaticamente!

### 5️⃣ Gerencie Despesas

1. Após criar o evento, clique em **Ver**
2. Clique em **Gerenciar Despesas**
3. Adicione custos (músicos, hotel, transporte, etc.)
4. Veja o **lucro líquido calculado automaticamente**

## 💡 Dicas Úteis

### 📊 Dashboard
- Os KPIs são calculados automaticamente para o mês atual
- O gráfico mostra os últimos 6 meses
- Alertas são atualizados a cada 5 minutos

### 🎤 Artistas
- Você pode pausar um artista sem deletá-lo
- Cada artista pode ter seu próprio modelo de contrato
- A comissão pode ser diferente por artista

### 🎪 Eventos
- Adicione todas as despesas para calcular o lucro real
- A equipe do artista é sugerida automaticamente
- Contratos são gerados no momento da criação

### 💰 Financeiro
- Parcelas atrasadas aparecem em vermelho
- Use os filtros para encontrar parcelas específicas
- O fluxo de caixa ajuda a identificar tendências

### 📄 Contratos
- Use variáveis como `{{nome_artista}}` no modelo
- Os contratos são preenchidos automaticamente
- Você pode editar o modelo a qualquer momento

## ❓ Perguntas Frequentes

### **Como adiciono mais artistas?**
Vá em **Artistas** → **Novo Artista**. O sistema suporta até 10 artistas.

### **Posso deletar um evento?**
Sim, se você for Admin Master ou Manager do artista vinculado.

### **Como funciona a comissão?**
A comissão é descontada do cachê bruto para calcular o valor líquido que o artista recebe.

### **O que são as variáveis do contrato?**
São placeholders como `{{nome_artista}}` que são substituídos automaticamente pelos dados reais do evento.

### **Como vejo o lucro de um evento?**
Acesse **Eventos**, clique em **Ver** e verifique as despesas. O lucro é calculado automaticamente.

### **Posso exportar relatórios?**
A funcionalidade de exportação está planejada para a versão 1.1.0.

## 🆘 Problemas Comuns

### Não consigo fazer login
- Verifique se digitou corretamente: `admin` / `admin123`
- Verifique o console do navegador (F12)

### Dados não aparecem
- Aguarde alguns segundos (carregamento)
- Atualize a página (F5)
- Verifique o console (F12) para erros

### Modal não abre
- Feche outros modals com ESC
- Atualize a página
- Verifique o console (F12)

### Gráfico não aparece
- Aguarde o carregamento completo
- Verifique se há dados para exibir
- Tente redimensionar a janela
